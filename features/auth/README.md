# МОДУЛЬ АВТОРИЗАЦИИ (AUTH FEATURE)

## Ключевая задача:
Реализовать бесшовный вход пользователя в приложение в один клик с использованием существующего Google-аккаунта на смартфоне (OAuth 2.0).

## Инженерные требования (Закон 0 вон):
1. Использовать Firebase Auth (Бесплатный тариф).
2. Пользователь НЕ должен вручную вводить логины, пароли или капчи.
3. Процесс должен происходить через системное всплывающее окно Android (Bottom Sheet).

## Логика работы UI/UX:
1. При первом запуске приложение проверяет состояние авторизации.
2. Если пользователь не вошел, отображается приветственный экран с единственной кнопкой "Войти с Google".
3. По нажатию кнопки вызывается системное окно выбора Google-аккаунтов.
4. После тапа на аккаунт приложение получает безопасный токен (ID Token), отправляет его в Firebase Auth и мгновенно открывает главный экран очков с Виолеттой.
