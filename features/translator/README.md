# МОДУЛЬ ПЕРЕВОДЧИКА (TRANSLATOR FEATURE)

## Ключевая задача:
Обеспечить быстрый и точный перевод (корейский <-> русский) с использованием движка Naver Papago, адаптированный для русскоязычных экспатов в Южной Корее.

## Инженерные требования (Закон 0 вон и Защита от банов):
1. ЗАПРЕЩЕНО использовать платное корпоративное API Naver Cloud Platform.
2. Использовать метод скрытых Web-Scraping HTTP-запросов к мобильному эндпоинту `https://naver.com`.
3. Запросы должны уходить НАПРЯМУЮ с мобильного устройства пользователя, используя его локальный IP-адрес. Это распределяет нагрузку и исключает блокировку.
4. Обязательно подменять заголовки (`Headers`), маскируя трафик под реальный мобильный браузер (динамический `User-Agent`).

## Режимы работы:
1. Голосовой синхронный перевод (Слушаем корейскую речь через микрофон -> выводим русский текст/озвучку в очки).
2. Запрос по требованию ("Виолетта, переведи [фраза]").
3. Интеграция с камерой (OCR) — отправка захваченного кадра на перевод в Papago Image Web-API при отсутствии Naver-аккаунта.
